    ONLINE VOTING SYSTEM
    Abstract:

    The project is mainly aimed at providing a secured and user friendly Online Voting System. The problem of voting is still critical in terms of safety and security. This system deals with the design and development of a webbased voting system using fingerprint and aadhaar card in order to provide a high performance with high security to the voting system. The proposed Online Voting System allows the voters to scan their fingerprint, which is then matched with an already saved image within a database that is retrieved from aadhaar card database of the government. The voting system is managed in a simpler way as all the users must login by aadhaar card number and click on his/her favorable candidates to cast the vote By using biometric fingerprint it provides enough security which reduces the dummy votes.
